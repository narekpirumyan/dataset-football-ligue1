# Visual Charter

**Scope:** General guidance for report visuals. No per-club color mapping is used in this project.

- **Club colours:** There is **no** dedicated colour assigned to each club. Charts and visuals use the default Power BI palette (or the report theme, e.g. `theme_arena_light.json`). Series and legends get colours from the theme in order of appearance.
- **Themes:** Apply the report theme (e.g. **Dashboard Design/theme_arena_light.json**) via **View → Themes** in Power BI for consistent backgrounds, data colours, and visual styles.
- **Club logos:** Where used, logo URLs are provided by `DimClub[Club_URL]` or **club_logos.json**; see dashboard specification for usage.

For detailed visual and page layout, refer to the dashboard specification and the design documents in **Dashboard Design/** (Club_View, Player_View, Ligue-Wide_View).
